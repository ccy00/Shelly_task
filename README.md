## Start

```bash
npm install

npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```

## Project structure

```
project/
├ src/
│ ├ application/    - core code
│ │ └ webhook/
│ │ └ worker/
│ ├ routes/
│ │ └ api/    - backend restful api
└ └ └ views/    - frontend page
```
